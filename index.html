Horizons


We take an idea from some words in a .doc file to a beautiful and functional website that your users will love.
This includes stuff like:

- Writing the specifications
- Creating mock-ups, Ui/ UX
- Mobile Ui / Ux and prototyping
- Creating a beautiful design for your site (in PSD format). We'll design something that fits with your company image and that your users will love
- Converting the design files in to clean HTML + CSS (PSD to HTML slicing)
- Writing the back-end code that makes your site work. We can also integrate 3rd party services like Stripe and Paypal for payments, Facebook, Twitter and Youtube for managing your online presence and even integration with your CRM (if you have one)
- Setting up the server for your site, to ensure that the launch goes smoothly
- Training for your staff on how to use the site
- Optional long term support for your site. Because we know that a site needs some changes to keep up with the competition. And we'll be happy to help.

We can also do anything in between. For example, if you just one some awesome designs, we can help. If you want to slice some PSDs, we can help. Need just the server side code written? Yes, we can help with that also.

Tools and technologies that we use and love:
Sketch 3 / Photoshop / Ilustrator / Affinity Designer to create designs.
HTML + CSS, Bootstrap and SASS to transform the designs from psd files to html.
Javascript, jQuery and AngularJS to make the site interactive.
Ruby on Rails or PHP/Laravel on the server.
MySQL or MongoDB for the database.
AWS, DigitalOcean or your own servers for hosting the site.
